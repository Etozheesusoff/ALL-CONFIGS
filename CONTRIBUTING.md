## üöÄ How Can I Contribute?

### 1. **Add New Configurations** üéÆ
Share optimized configs for games or GPUs.

**Steps:**
1. Create a new folder: `GameName/configs/GPU-Model/level-name/`
2. Add your config files (with comments)
3. Create a README describing your config
4. Submit a Pull Request with benchmark results

**Template:**
```
Config Name: [Game] [GPU] [Level]
Author: @YourUsername
Benchmark Results:
- Average FPS: XXX
- Min FPS: XXX
- Max FPS: XXX
Hardware: [Your PC specs]
```

### 2. **Improve Documentation** üìö
- Fix spelling/grammar
- Add new guides
- Translate to other languages
- Clarify existing docs
- Add examples

### 3. **Report Bugs** üêõ
Found an issue? Please report it!

**When reporting:**
- Use the Bug Report template
- Describe reproduction steps
- Include system info
- Attach screenshots/logs
- Link to related issues

### 4. **Suggest Features** üí°
Have an idea? Share it!

**When suggesting:**
- Use Discussions first
- Check if it's already planned
- Explain use case
- Provide examples

### 5. **Help Others** üôã
- Answer questions in Discussions
- Help solve issues
- Test configs
- Provide feedback

## üõ†Ô∏è Development Setup

### Prerequisites
- Git
- Python 3.6+
- Your favorite code editor
- The game(s) you're configuring

### Setup
```bash
# Clone the repository
git clone https://github.com/Etozheesusoff/ALL-CONFIGS.git
cd ALL-CONFIGS

# Create a branch
git checkout -b feature/your-feature-name

# Make your changes
# ... edit files ...

# Test your changes
# ... verify configs work ...

# Commit
git add .
git commit -m "type: description

Longer explanation if needed."

# Push
git push origin feature/your-feature-name
```

### Commit Message Format

```
type: subject

body (optional)

footer (optional)
```

**Types:**
- `feat:` New configuration or feature
- `fix:` Bug fix
- `docs:` Documentation changes
- `style:` Code style changes
- `refactor:` Code refactoring
- `perf:` Performance improvements
- `test:` Test additions
- `chore:` Maintenance tasks

**Examples:**
```
feat: Add RTX 4090 high-end CS2 config

feat: Add Valorant low-end AMD configs

docs: Update installation guide for Windows

fix: Correct typo in FAQ

perf: Optimize CS2 config for 240 FPS
```

## üìù Pull Request Process

### Before Submitting

1. **Test Your Config**
   - [ ] Tested on real hardware
   - [ ] Benchmark results collected
   - [ ] No crashes or errors
   - [ ] Performance improvement verified

2. **Follow Style Guidelines**
   - [ ] Config files have comments
   - [ ] File names follow conventions
   - [ ] README is clear and detailed
   - [ ] No sensitive information included

3. **Update Documentation**
   - [ ] Updated relevant READMEs
   - [ ] Updated CHANGELOG.md
   - [ ] Added to comparison tables
   - [ ] Created/updated guides if needed

4. **Code Quality**
   - [ ] No syntax errors
   - [ ] Consistent formatting
   - [ ] Clear and concise comments
   - [ ] No duplicate content

### PR Template

```markdown
## Description
Brief description of changes

## Type
- [ ] New config
- [ ] Bug fix
- [ ] Documentation
- [ ] Other

## Game & GPU
- Game: [CS2/Valorant/OW2]
- GPU: [NVIDIA/AMD/Intel Model]
- Level: [Low-end/Mid-range/High-end]

## Benchmark Results
- Average FPS: XXX
- Min FPS: XXX
- Max FPS: XXX
- Previous: YYY FPS (+X%)

## Hardware Used
- CPU: 
- RAM: 
- GPU: 
- Motherboard: 

## Testing
- [ ] Tested on real hardware
- [ ] No crashes
- [ ] Performance improvement verified
- [ ] Works with latest game version

## Related Issues
Fixes #123
Related to #456

## Screenshots/Logs
[Attach if relevant]
```

### PR Review Process

1. Automated checks will run
2. Maintainers will review
3. You might be asked for changes
4. Once approved, your PR will be merged
5. You'll be added to CONTRIBUTORS.md

## üêõ Bug Report Guidelines

### Title
Be specific and descriptive.

‚ùå Bad: "Config doesn't work"
‚úÖ Good: "RTX 4090 high-end CS2 config causes crashes on Nuke"

### Description
Include:
1. What you expected to happen
2. What actually happened
3. Steps to reproduce
4. Your system info
5. Game version
6. Config version
7. Any error messages

### Attachments
- Screenshot of error
- System info output
- Game log file
- Benchmark results

## üìà Suggestion/Feature Request Guidelines

### Title
Clear and concise.

‚ùå Bad: "New thing"
‚úÖ Good: "Add support for Arc A380 low-end configs"

### Description
Include:
1. Problem statement (Why is this needed?)
2. Proposed solution
3. Benefits
4. Alternative approaches considered
5. Any concerns

### Examples
Show real-world use cases with examples.
