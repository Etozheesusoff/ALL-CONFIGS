# Intel High-end — CS2 Config

Данный пресет предназначен для систем с современными Intel iGPU или дискретными картами Intel Arc, когда CPU и GPU не являются узким местом.

- Используйте как базу и подстраивайте:
  - разрешение,
  - качество текстур,
  - сглаживание.

Рекомендуется:
- протестировать несколько матчей;
- при необходимости перейти к настройкам из `docs/guides/optimization/cs2-guide.md`.
