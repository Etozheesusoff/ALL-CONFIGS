# **NVIDIA High-end — CS2 Config**

**Цель:** максимум плавности и отзывчивости на топовых системах NVIDIA.

**Рекомендуется для:**
- RTX 3070 / 3080 / 4070 / 4080 / 4090
- 16–32 GB RAM
- 144–240+ Hz мониторы

**Файлы:**
- `autoexec.cfg` — сбалансированные сетевые и игровые настройки
- `cs2_video.txt` — 1440x1080p, средние/низкие эффекты

**Установка:**

* Вашу видеокарту (NVIDIA / AMD / Intel);
* Уровень железа (low / mid / high).

**Скопируйте:**

* autoexec.cfg → в папку конфигов CS2 `C:\\Program Files (x86)\\Steam\\steamapps\\common\\Counter-Strike Global Offensive\\game\\csgo\\cfg`;

* cs2_video.txt → в папку с пользовательскими настройками `C:\\Program Files (x86)\\Steam\\userdata\\ВАШ\\\_ID\\730\\local\\cfg`.

> Если у Вас не один ID в папке `userdata` - то смотрите по дате изменения, самый свежий точно Ваш.

Пути и команды подробно описаны в CS2/README.md.
