## üá∫üá∏ English

### 1. Where to find your settings?
1. Press `Win + R` and paste: `%LOCALAPPDATA%\VALORANT\Saved\Config`
2. Open the folder with a long string of random characters (check the "Date modified").
3. Go to the `Windows` folder. Your config file is **`GameUserSettings.ini`**.

### 2. How to install
1. Download `GameUserSettings.ini`.
2. Go to the path mentioned above (your `Windows` folder).
3. Replace the old file with the downloaded one.
4. **Important:** Right-click the file -> **Properties** -> check **"Read-only"**. This prevents the game from overriding your settings.

---
> **üí° Tip:** To get more FPS, you can manually change `sg.ResolutionQuality` to `80.000000` inside the file.
